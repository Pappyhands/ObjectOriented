****How to Compile and Run a Java File using C9.io****

LINK: https://www.digitalocean.com/community/tutorials/how-to-install-java-with-apt-get-on-ubuntu-16-04


While in the Directory w/ the Source Java File:
-------------------------------------------------
1.) To Compile:
    javac (FILENAME)

2.) To Run:
    java (FILENAME)
    
    

ex. cd HelloWorld\
    javac HelloWorld.java
    java HelloWorld.java
    
This should run the program.
